# シャカイ・クエスト（仮）ノードマップ設計書 v0.1

理科版「リカ・クエスト」の後継として、東京書籍「新編 新しい社会」（令和6年度版）5・6年の全単元を、時空クロニクル×あつ森×桃鉄型の探検ゲームに落とし込むための全ノード一覧です。

**注意**：収蔵品・偉人候補は令和6年度版の年間指導計画（単元一覧表）から起こした設計用の例示です。実際の教科書本文にある具体的なモデル地域・写真・資料とは異なる場合があるので、実装時は教科書原本で要確認。歴史上の人物は学習指導要領で例示されている人物（卑弥呼・聖徳太子・源頼朝・織田信長など）を中心に選定しています。

## 凡例

- **ID規則**：`s{学年}_{分野略号}{連番}`（例：s5_koku01＝5年・国土ライン1駅目）
- **難易度3段階**：基本＝探検／発展＝認定チャレンジ／中学生＝もっと知りたい！チャレンジ
- **収蔵品カテゴリ**：名産品・アイテム＝資料館に並ぶ図鑑枠、認定キャラ／偉人＝認定チャレンジ突破でもらえるカード枠
- **分岐**：教科書側がもともと2択にしている小単元はルート分岐として扱う

## 設計メモ

- リカ・クエストの既存ルールを継承：基本／発展の出題範囲は小単元・指導要領の範囲内に限定し、中学生範囲の用語や発展トリビアは「もっと知りたい！チャレンジ」枠のみに出す
- s6_rek11（長く続いた戦争と人々のくらし）だけ演出を変更。認定キャラとの問答ではなく、証言・資料を読み解く形式にする
- 歴史上の人物のセリフは「〜と伝えられている」「〜という業績を残した」のようなナレーション中心にし、断定的な創作名言は避ける
- 5年の「認定キャラ」は全て架空の役職名（名人・案内人など）。実在の人物名は使わない
- 政治・国際ラインの案内人も架空キャラで統一し、実在の現職政治家は登場させない
